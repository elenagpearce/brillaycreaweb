Generador de Destellos

⚠️ Importante — cómo funciona realmente esta herramienta Este sitio es 100% estático (vive en GitHub Pages), no tiene servidor ni base de datos. Esta página no guarda nada por sí sola: te ayuda a redactar y ordenar el contenido, y al final te da un bloque de código para copiar y pegar tú misma en el archivo destellos-data.js del repositorio. La imagen tampoco se sube desde aquí — la seleccionas solo para ver la vista previa, y por separado la subes a la carpeta /destellos/ del repositorio con el mismo nombre que escribas abajo.

Esta clave de acceso es solo un filtro simple para que no cualquiera que llegue a esta URL la use — no es seguridad real (cualquiera que vea el código fuente podría encontrarla). No compartas este link públicamente y no lo enlaces desde el menú del sitio.
